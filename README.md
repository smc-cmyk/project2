
# CST338 Project 2: Currency Converter

Currency Converter displays current and historical exchange rates, displays a graphical representation of historical exchange rates, and converts a user’s input from USD to another. 
This application will allow the user to login and navigate through different options in the main menu. 
In the main menu the user has the option to navigate through the USD converter, view current exchange rates, view historical exchange rates in a table view, view historical exchange rates in a graphical view, view the instructions for the application, or logout of the application.

## Features
- User login and registration with information stored in a database
- Instruction scene demonstrating how to use the application
- Table displaying current exchange rates from USD to supported currencies
- Convert USD to our supported currencies (Euro, Yen, British Pound, Chinese Yuan, and Canadian Dollar)
- Receive a breakdown of your conversion and date of data retrieval
- Table displaying historical exchange rates from 1970 to 2026
- Graph displaying historical exchange rates from 1970 to 2026
